TexasHoldem — Module
TexasHoldem

A no-limit Texas Holdem simulator.

Terminology

  • game a single "game", where players are dealt hands, winner(s) are declared once.
  • stage a point or process in the game, including PreFlop, Flop, Turn, River.
  • round the process of each player deciding which actions to take, until no further actions are taking.

Raise bet, for the round, to amount amt. Example:

# Flop
Player[1] raise to 10 (`amt = 10`, contribute 10)
Player[2] raise to 20 (`amt = 20`, contribute 20)

Player[2] raise to 2 (`amt = 2`, contribute 2)
